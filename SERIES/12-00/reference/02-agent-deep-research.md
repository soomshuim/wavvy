## 1. Rhythm & Percussion (리듬 & 퍼커션)

### 1.1 핵심 리듬 패턴

**Clave 패턴**
- **3-2 / 2-3 Son Clave** — 카우벨/우드블록이 곡 전체를 관통하는 "가이드 리듬"
- Highlife 기타 아르페지오 = **3-2 clave motif guajeo**와 동일 attack-point 구조

**Dembow 변형**
- Afrobeats와 Dembow은 공통 조상 공유
- 핵심: **3+3+2 (Tresillo)** 크로스리듬 + 오프비트 싱코페이션
- 가나 Gahu, 콩고 Mbilu a Makinu에서 기원

**폴리리듬**
- 드럼 4/4 유지 + 콩가/토킹드럼은 독립 패턴
- 서로 다른 박자가 겹쳐 입체적 사운드

### 1.2 퍼커션 악기 역할

| 악기 | 역할 | 특징 |
|------|------|------|
| Djembe | 리듬 백본 | 깊고 파워풀, 안정적 펄스 |
| Talking Drum | 리드 퍼커션 | 피치 변화, Call & Response |
| Shekere | 필러/무브먼트 | 드럼 사이 공간 채움 |
| Congas | 싱코페이션 레이어 | 복잡한 패턴 |
| Cowbell | 타임키퍼 | 기본 리듬 유지 |
| Agogo | 싱코페이션 악센트 | syncopated 패턴 |

### 1.3 드럼 프로그래밍

- **Kick**: 1/3박 펀치, 싱코페이션 변주
- **Snare/Clap**: 2/4박 백비트
- **Hi-Hat**: 16분음표, 오프비트 오픈

**레이어링 순서:**
1. Kick + Snare
2. Hi-hat 16분음표
3. Shaker/퍼커션 필러
4. Conga/Bongo
5. Cowbell/Agogo 악센트

### 1.4 BPM별 서브장르

| 서브장르 | BPM | 특징 |
|----------|-----|------|
| Afrobeats (일반) | 95-120 | 메인스트림, 댄서블 |
| Amapiano | 108-115 | 무디, 로그드럼 중심 |
| Afropop | 122-135 | 업템포, 댄스 |
| Afrosoca | 110-135 | 서아프리카+소카 |
| Street-Hop | 120+ | 에너지틱, 래핑 |
| Alte | 다양 | 실험적 |
| Afro-House | 120-130 | 하우스+아프리카 퍼커션 |

### 1.5 Log Drum
- Amapiano 시그니처: 퍼커시브 나무 같은 딥 베이스
- Suno: `Log Drum bassline` 또는 `Deep percussive bass`

## 2. Instruments & Sound Design

### 2.1 핵심 악기

| 악기 | 역할 | 사운드 |
|------|------|--------|
| Electric Guitar | 하모닉/멜로디 | Highlife 핑거피킹 |
| Bass/808 | 그루브 파운데이션 | 싱코페이션, 킥 동기화 |
| Rhodes/Wurlitzer | 하모닉 텍스처 | 따뜻, 스타카토 오프비트 |
| Synth/Pads | 공간감 | 루시 코드, 플러키 리드 |
| Saxophone | 멜로디 리드 | 감성 솔로 |
| Brass | 에너지/파워 | 트럼펫+테너색소폰 |
| Percussion Kit | 리듬 레이어 | 셰이커, 콩가, 봉고 |

### 2.2 Highlife Guitar
- **투핑거 플러킹**: Kru 선원 "팜와인 기타" 기법
- **Guajeo 아르페지오**: 3-2 clave motif 기반
- **하모닉**: 3rd(mediant) + 4th(subdominant) 활용
- **최면적 순환**: 멜로딕 베이스라인 + 타이트한 킥 퍼커션

### 2.3 베이스라인
- 싱코페이션이 핵심, 킥과 locked-in
- 808: 킥 주변을 노는(play around) 배치
- 단순함과 움직임의 균형
- 유형: 긴 808 vs 짧은 808, 루트만 vs 코드 루트 따라감

### 2.4 신스/패드
- Rhodes/Wurlitzer 기본
- 2~4개 반복 코드, 펜타토닉/도미넌트 7th
- 짧은 스타카토, 오프비트 히트
- 기본 3화음 + 메이저 6th/7th 추가

### 2.5 프로듀서 시그니처

| 프로듀서 | 특징 | 대표 작업 |
|----------|------|----------|
| **Sarz** | 넓은 스펙트럼, EDM~전통 | Wizkid, Tems, Asake |
| **P2J** | 아프리카를 모든 것에 녹임, 세련됨 | Afro+R&B/House/Pop |
| **Tempoe** | "MAD!" 태그, Afro-fusion | CKay "Love Nwantiti" |
| **London** | 무디 트랩+Afrobeat 퍼커션 | Rema "Ozeba", Ayra Starr |

### 3.3 훅 작법
- **3~8 단어 반복** (군중이 따라 부를 수 있는 길이)
- **4단어 코러스** — 가장 효과적
- **링 프레이즈** — 제목이 시작과 끝에 반복
- **마지막 2단어 태그 반복** — 기억 강화
- **Call & Response** — 호응 구조
- "챈트가 우연히 아름다워진" 느낌이 최고의 훅

### 3.4 한국어 Afrobeats 접근법
1. 겸손과 호기심으로 접근
2. 현지 표현을 소품으로 쓰지 않기
3. 한국어 + 영어 + Pidgin 소량 믹스
4. 톤 언어 주의 (요루바/이그보 = 성조 언어)
5. 한국어 음절의 자연스러운 싱코페이션 활용
6. PENOMECO "ORGANIC" 선례 참고

### 4.1 서브장르 비교

| 항목 | Afrobeats | Amapiano | Afro-Fusion | Alte |
|------|-----------|----------|-------------|------|

| 핵심 사운드 | 퍼커션+캐치 멜로디 | 로그드럼+딥 베이스 | 장르 혼합 | Alt R&B+댄스홀+힙합 |

### 4.2 2026 트렌드 — "Afrobeats 3.0"
- 슬로우번/미드템포가 라우드 클럽 뱅어를 스트리밍에서 앞지름
- "헤비 드럼+캐치 훅+댄스 챌린지=히트" 공식 소멸
- **의도적 레코드** — 15초 바이럴에서 벗어남, 사운드 아이덴티티 추구
- **앨범 르네상스** — 코히전트 앨범 다시 중요
- **로컬 루츠 회귀** — Fuji, Highlife 전통 요소 재발견

### 4.3 글로벌 크로스오버 성공 요인

## 5. Suno Production

### 5.1 Style Prompt 키워드

**구조:** `[장르 태그], [템포], [보컬 스타일], [핵심 악기], [분위기], [BPM]`

| 스타일 | 프롬프트 |
|--------|---------|
| Burna Boy | `Afrobeats, Afro-fusion, mid-tempo, smooth male vocals, saxophone melody, complex percussion, mellow summer vibe, 95 BPM, minor key` |
| 클래식 | `Afrobeat, upbeat, 105 BPM, syncopated percussion, djembe, live horns, funk guitar` |
| Afro-Fusion | `Afrobeats fusion, rhythmic percussion, smooth female vocals, bright synth leads, danceable warm tone, 105 BPM` |
| Amapiano | `Amapiano, Log Drum bassline, deep percussive bass, spacious arrangement, smooth vocals, 112 BPM` |
| K-pop 퓨전 | `Afrobeats, K-pop, syncopated percussion, bright synths, catchy hook, polished production, danceable, 106 BPM` |
| 한국어 | `Afrobeats, Korean Pop, mid-tempo, smooth vocals, African percussion, guitar riff, warm bass, 105 BPM` |

**Suno 장르 태그:**
`afrobeat`, `afrobeats`, `afropop`, `afroswing`, `afro r&b`, `afro-funk`, `afrobeat fusion`, `amapiano`, `highlife`, `nigerian pop`

### 5.2 이펙트

**Reverb:**
- 리드 보컬: 5-10% (매우 적게)
- 패드/보컬: 긴 리버브 테일로 공간감
- 리버브 테일에 하이패스 필터 → 클린+깊이

**Delay:**
- 리드 보컬: 5-15%
- 킥/베이스에 딜레이 절대 금지
- 핑퐁 딜레이로 리듬적 상호작용

### 5.3 믹싱
- 각 악기가 자기 주파수 공간 확보 (EQ)
- 드럼/베이스: 센터 / 퍼커션/기타: 좌우
- 넓은 스테레오 이미지

### 5.4 Exclude 프롬프트
```
Exclude: heavy distortion, metal guitar, orchestral strings, EDM buildup,
aggressive rap, country twang, harsh noise, no autotune, no EDM drop
```
