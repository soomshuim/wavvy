## 1. 리듬 & 퍼커션

### 핵심 리듬 DNA
- **3-2 / 2-3 Son Clave** — 곡 전체를 관통하는 가이드 리듬
- **Tresillo (3+3+2)** — 킥/하이햇 배치의 기본 뼈대, Dembow과 공통 조상
- **폴리리듬** — 4/4 드럼 위에 콩가/토킹드럼이 독립 패턴

### 퍼커션 악기

| 악기 | 역할 | Suno 키워드 |
|------|------|------------|
| Djembe | 리듬 백본 | `djembe` |
| Talking Drum | 리드, 피치 변화 | `talking drum` |
| Shekere | 드럼 사이 필러 | `shaker`, `shekere` |
| Congas | 싱코페이션 레이어 | `congas` |
| Cowbell | 클라베/타임키퍼 | `cowbell` |
| Log Drum | Amapiano 딥 베이스 | `log drum bassline` |

### 드럼 프로그래밍 (5단계 레이어링)
1. Kick (1/3박) + Snare/Clap (2/4박)
2. Hi-hat 16분음표 (오프비트 오픈)
3. Shaker/Shekere 필러
4. Conga/Bongo 레이어
5. Cowbell/Agogo 싱코페이션

## 3. 화성 & 코드

### 원칙
- **3화음 중심** — 재즈틱하면 장르 이탈
- 코드는 **"지탱"** 역할 (그루브 방해 금지)
- **펜타토닉에 가까운 음재**
- 패드: 3화음+add9 정도만

### 코드 진행

| 아키타입 | 진행 | 용도 |
|---------|------|------|
| 팝 메이저 | I–V–vi–IV | 코러스/후렴 |
| 팝 마이너 | vi–IV–I–V | 벌스/감정선 |
| 2코드 최면 | IV–I | 루프/배경 |
| 반음 인접 | I ↔ 반음 아래 | 아프리카적 색채 |

## 4. 멜로디 & 보컬

### 설계 원칙

| 요소 | 패턴 | Suno 키워드 |
|------|------|------------|
| 프레이즈 | 1-2마디 반복+변주 | `1-2 bar repeating vocal motif` |
| 음정 | 2도·3도 계단식 | `stepwise melody` |
| 음역 | 1옥타브 내 | `compact range` |
| 장식 | 말미 2-3음 슬러 | `short slurred endings` |
| 종지 | 약간 열린 (루프용) | `loop-friendly cadence` |
| 콜앤리스폰스 | 리드+백업/샤우트 | `call-and-response` |

### 보컬 톤
- warm, conversational, chest voice
- R&B 런 길어지면 장르 이탈
- 짧은 슬러/글리산도가 장르감

## 6. 구조 & 편곡

### 트랙 구조 (3-4분)
```
Intro(4-8) → VerseA(8) → VerseB(8, C&R) → Pre(4-8) → Chorus(8-16)
→ VerseA'(8) → Chorus'(8-16) → Outro(4-8)
```

### 핵심
- 벌스 내부 A/B/C 모듈 분절
- 리프레인 반복 회귀 (론도 유사)
- Outro: 그루브만 남기고 크로스페이드 포인트

## 9. Suno 프로덕션 종합

### Style Prompt 구조
```
[장르 태그], [템포], [보컬 스타일], [핵심 악기], [분위기], [BPM]
```

### 장르 태그 옵션
`afrobeats`, `afropop`, `nigerian pop`, `afrobeat fusion`, `afro r&b`, `amapiano`, `highlife`, `afroswing`, `afro-funk`

### Exclude 템플릿
```
EDM drops, heavy distortion, metal guitar, orchestral strings,
falsetto, whisper vocals, aggressive rap, autotune heavy,
country twang, harsh noise
```

### 이펙트 가이드
- Reverb: 리드 보컬 5-10%, 패드는 길게
- Delay: 리드 5-15%, 킥/베이스 절대 금지
- 믹싱: 드럼/베이스 센터, 퍼커션/기타 좌우 패닝
