# 12-00 Korean Afrobeats — 가사 DNA

> Version: 1.0 | Source: 가사 딥리서치 (2026-03-13) + 종합 리서치 (2026-03-11)
> Purpose: 가사 작성 시 이 문서만 참조. 리서치 원문 직접 참조 불필요.

---

## 1. 톤/어투

- **자신감+축제적** (celebratory) — Afrobeats의 기본 톤
- **1인칭** (자부심+취약성 혼합): "나는 ~해" "내가 ~할게"
- **2인칭** (직접 호칭, 친밀감): "너 ~해봐" "네가 ~하면"
- **명령형** (행동 촉구, 파티 유도): "털어" "던져" "흔들어"
- **현재 시제 우선**, 관찰자 아닌 **참여자** 시점
- 추상 감정 대신 **구체적 장면** 묘사 ("가로등이 번져" 식의 사물+동작)

---

## 2. 훅 공식

- **3~8음절** 초단문 (한 번 듣고 따라 부를 수 있는 길이)
- **Ring Phrase**: 코러스 1행 = 마지막 행 (원형 반복)
- **비언어적 가능**: 의태어 반복, 감탄 챈트 ("들썩들썩", "둥둥")
- **스냅샷형**: 명사+동사 한 컷 ("발바닥이 달아" / "연기가 피어")
- **스네어 착지**: 핵심 음절을 강박(2/4박)에 배치

---

## 3. 코드스위칭 = 의태어

Afrobeats에서 Pidgin English/Yoruba가 "감정적 구두점" 역할 →
**한국어에서는 의태어/의성어가 동일 기능 수행**

| Afrobeats 원본 | 한국어 대응 | 기능 |
|---------------|-----------|------|
| "Ye ye ye" | 의태어 반복 | 감정 폭발 포인트 |
| "Oya!" | "자!" "어!" | 행동 전환 신호 |
| Pidgin 삽입 | 의태어 삽입 | 리듬 변화+감정 강화 |
| Ad-lib chant | 의성어 챈트 | 퍼커시브 보컬 |

**규칙:**
- 감정 강화 포인트에서만 (남발 금지)
- Verse당 1개+ 의태어/의성어 필수
- Post-chorus에 챈트형 의태어 반복 구간 설계

---

## 4. 반복 기법

| 기법 | 설명 | 한국어 적용 |
|------|------|-----------|
| **Phrase Looping** | 같은 구절 2~3회 + 미세 변형 | "털어 털어 다 털어" |
| **Ring Phrase** | 코러스 시작=끝 | 1행과 마지막 행 동일 |
| **Post-chorus 챈트** | 단순 음절 반복 (군중 참여) | 의태어 2~4음절 반복 |
| **최면적 반복** | 짧은 모티프가 곡 전체 반복 | 핵심 이미지 1개가 Verse/Chorus 관통 |

---

## 5. 이미지 소재 풀 (4 클러스터)

> **각 Verse에 최소 2개 클러스터 교차 필수**

| 클러스터 | 단어 뱅크 | 감각 |
|---------|----------|------|
| **신체/동작** | 들썩, 흔들, 털어, 꿈틀, 구르, 뻗어, 밟아, 구부려, 젖혀, 감아 | 운동감각 |
| **음식/감각** | 볶아, 달궈, 끓여, 씹어, 익어, 타, 연기, 김, 기름, 냄새 | 미각+후각+촉각 |
| **해방/에너지** | 터져, 날려, 풀려, 던져, 비워, 깨, 벗어, 쏟아, 뜯어, 찢어 | 해방감 |
| **리듬/소리** | 쿵쿵, 둥둥, 탁탁, 딱딱, 찰칵, 쩌렁, 와르르, 쏴아 | 청각 |

**소재 교차 예시:**
- 신체+음식: "발바닥이 달아" (신체의 열감 = 음식 굽는 감각)
- 음식+해방: "연기가 터져" (조리 과정 = 에너지 방출)
- 신체+리듬: "어깨가 둥둥" (신체 움직임 = 소리)

---

## 6. Anti-Pattern (시티팝 금지)

> **3회 실패 원인. 1건이라도 발견 시 즉시 재작성.**

| 시티팝 함정 | Afrobeats 대체 |
|------------|---------------|
| 네온, 가로등, 도시의 밤 | 햇빛, 땀, 먼지, 시장, 불판 |
| "~처럼 흘러가" 류 비유 | "털어" "던져" "볶아" 동작동사 |
| 몽환적/회고적 톤 | 축제적/현재 시제/지금 여기 |
| 3인칭 관찰자 시점 | 1인칭 참여자 + 2인칭 호출 |
| 추상적 감정 (그리워, 외로워, 아련해) | 신체 반응 (들썩, 끓어, 터져, 달아) |
| 수동태/피동형 (번져, 젖어들어) | 능동태/명령형 (털어, 던져, 깨) |
| 고요/정적 이미지 (멈춰, 가라앉아) | 움직임/동적 이미지 (구르, 뻗어, 밟아) |

---

## 7. 콜앤리스폰스 설계

- **리드 1줄** → **백킹 짧은 응답** (의태어/감탄 1-2음절)
- Verse에서도 활용 (코러스만이 아님)
- Suno 가사 입력 시 `(ad-lib)` 또는 `(call-and-response)` 괄호 표기

**패턴 예시:**
```
[verse]
(Chest voice, Direct)
발바닥이 달아 (에!)
연기가 올라와 (에!)
```
