## Concept

> **"한낮의 그루브 — 몸이 먼저 움직이는 점심"**

12시. 일에서 잠시 벗어나는 시간.
배가 고프고, 몸이 가볍고, 리듬이 앞서는 순간.
Afrobeats의 반복적인 훅과 퍼커션이 만드는 자연스러운 바운스.

### 2026 트렌드 반영 (Afrobeats 3.0)
- **미드템포 중심** — 라우드 클럽 뱅어 대신 바이비한 그루브 (100-112 BPM)
- **의도적 레코드** — 15초 바이럴용이 아닌, 사운드 아이덴티티가 있는 곡
- **로컬 루츠** — Highlife 기타, 전통 퍼커션 요소를 현대적으로 재해석

### 키워드
- 시간대: 점심, 한낮, 낮의 에너지
- 공간: 거리, 카페 테라스, 푸드트럭, 옥상
- 감정: 가벼움, 그루브, 기대, 여유로운 활력
- 동작: 걸음, 끄덕임, 손가락 탭, 어깨 흔들림

### Chorus 설계 공식

```
[Chorus]
{스냅샷 이미지 4-6음절} ← 훅 (링 프레이즈 시작)
{동작/감각 4-6음절}
{스냅샷 이미지 4-6음절}
{훅 반복 = 1행과 동일} ← 링 프레이즈 끝
```

예시 (방향 참고용, 실제 트랙과 무관):
```
[Chorus]

그림자가 따라와
어깨가 흔들려
발끝이 먼저 가
```

### Verse 설계 공식

```
[Verse]
{공간/시간 이미지} ← 장면 설정
{감각 디테일}
{움직임/동작}
{감각 디테일 변주}
```

**5-6행 확장 시:**
```
[Verse]
{공간/시간 이미지}
{감각 디테일}
{움직임/동작}
{감각 디테일 변주}
{리듬/소리 이미지} ← 추가
{동작 마무리} ← 추가 (에스컬레이션 시작점)
```

- V1-V2: Metric Mirroring (음절수/행수/문장구조 동일)
- V2 마지막 2행: 에스컬레이션 (동사 강화)
- 섹션당 4-6행, 행당 6-8음절 (LYRICS §2.7)

### Song Structure Pattern

**기본:** `V-C-V-C-B-C` (Pop Standard — LYRICS §2.5)
- Afrobeats 모듈형 구조와 가장 호환
- Verse 내부에서 장면 전환으로 A/B 분절감 연출
- Bridge는 에너지 유지 (드롭 금지, STYLE §3)

**변형 허용:** `C-V-C-V-C` (Short & Sweet) — 훅 먼저 시작하는 Afrobeats 관습 반영 시

### Bridge 설계 공식

```
[Bridge]
(Chest voice, stronger dynamics)
{Thesis — 현상 기반 고정 1행} ← B1/B2 동일
{Scene — 동작/감각 이미지 2행} ← B1/B2에서 변주
```

- **에너지 유지 필수** — Bridge에서 에너지 드롭 금지 (STYLE §3)
- Thesis: 시리즈 컨셉 관통하는 한 문장 (예: "거리가 흔들려")
- Scene: 감각 강도를 Verse보다 한 단계 올림
- Bridge → Chorus 연결: higher register + stronger dynamics
