## 시리즈 컨셉

### 핵심 아이디어
"밤산책"은 하나의 여정이다. 집 현관을 나서는 순간부터 다시 집으로 돌아오기까지, 10개의 장소를 지나며 밤의 공기를 마신다. 화자보다 **공간**이 주인공이다. 각 트랙은 그 장소의 분위기, 불빛, 소리, 온도를 음악으로 번역한다.

### 감정 아크
```
현관(설렘) → 골목(익숙함) → 편의점(활기) → 공원(사색) → 강변(해방)
    → 다리(전환) → 정류장(기다림) → 카페(온기) → 옥상(시야) → 귀가(안도)
```

### BPM 아크
```
75 → 88 → 108 → 72 → 115 → 92 → 68 → 85 → 112 → 70
     ↑              ↑                          ↑
   상승           피크                        피크2
```

## 장르 DNA (3축)

### 축 1: 클래식 시티팝 (3곡)
- **BPM**: 100-120
- **레퍼런스**: 유키카 NEON, 김현철 Drive/달의 몰락
- **특징**: 펑키 기타, 신스 패드, 댄서블 그루브
- **Key 경향**: F Major, A Major, G Major

### 축 2: 시티팝풍 R&B 발라드 (4곡)
- **BPM**: 65-80
- **레퍼런스**: 죠지 바라봐줘요
- **특징**: Rhodes/Felt Piano, 확장 코드(M9, 13, sus4), 느린 템포
- **Key 경향**: Db Major, Eb Major

### 축 3: 네오소울 + 시티팝 (3곡)
- **BPM**: 80-100
- **레퍼런스**: 아소토유니온 Think About Chu, 죠지 오랜만에
- **특징**: 재즈 코드, 그루비한 베이스, 따뜻한 톤
- **Key 경향**: Eb Major, Db Major

## 레퍼런스 분석 요약

### 죠지 - 바라봐줘요
- **BPM**: 67 | **Key**: Db Major
- **코드**: Dbadd9, GbM9, B13, Ab9sus4, Aaug 등 확장 코드
- **DNA**: 90년대 시티팝풍 R&B 발라드, Rhodes-led, 느린 템포

### 죠지 - 오랜만에
- **BPM**: 94 | **Key**: Eb Major
- **코드**: EbM7-Cm7-BbM7-Fm7
- **DNA**: 재즈/소울 시티팝, 따뜻한 7th 코드

### 유키카 - NEON
- **BPM**: 114 | **Key**: F Major
- **코드**: Gm7-C7-F-D7
- **DNA**: 현대 시티팝, 댄서블, 신스 기반

### 김현철 - 달의 몰락
- **Key**: A Major
- **코드**: Bm-C#m-F#m-D
- **DNA**: 클래식 한국 시티팝, 감성적 멜로디

### 아소토유니온 - Think About Chu
- **Key**: Db Major
- **코드**: C7-Am7-A7-Dm7
- **DNA**: 네오소울 R&B + 시티팝 감성

## Style Prompt 공통 요소

### Vocal Baseline
```
Raw vocal, Solid, Direct, Dry, Unprocessed
Chest voice dominant. No falsetto.
Precise articulation, clear consonants.
```

### Exclude 기본
```
Airy, Falsetto, Whisper, Harmonized, Backing vocals, Choir, Doubled vocals
```

### 장르별 Exclude 추가
- **시티팝**: Heavy, Dark, Aggressive, Slow ballad
- **발라드**: Energetic, Dance, Fast, Synth-heavy
- **네오소울**: Cold, Harsh, EDM, Auto-tune
