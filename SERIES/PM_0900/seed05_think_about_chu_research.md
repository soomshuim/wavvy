## 2. 코드 진행 분석 (악보 기반)

### 악보 표기 (Capo 1, C 셰이프)

```
[Intro / Verse / Chorus 공통 루프]
| Cadd9  C  CM7 | A7sus4  A7 | Dm  Dm7 | G5  Gsus4  G |
```

### 실제 울리는 코드 (Capo 1 → Db Major)

```
| Dbadd9  Db  DbM7 | Bb7sus4  Bb7 | Ebm  Ebm7 | Ab5  Absus4  Ab |
```

### 로마 숫자 분석 (Db Major)

```
| Iadd9  I  IM7 | VI7sus4  VI7 | IIm  IIm7 | V5  Vsus4  V |
```

### 핵심 화성 특징

| 특징 | 설명 |
|------|------|
| **원-루프 구조** | Intro/Verse/Chorus 모두 동일 4마디 루프 반복 |
| **확장 코드** | add9, M7, sus4, m7 — 재즈/네오소울 어법 |
| **Secondary Dominant** | Bb7 (VI7) = V7/IIm → Ebm으로 해결. 써클 오브 피프스 흐름 |
| **Voice Leading** | Cadd9→C→CM7 (실제: Dbadd9→Db→DbM7) — 최상단 음이 한 음씩 하강하는 부드러운 보이스 리딩 |
| **sus4 해결** | A7sus4→A7, Gsus4→G — 텐션 후 해결의 반복 패턴 |
| **주법 (스트럼)** | 싱코페이션 리듬 패턴 (근음 + 컷 + 뮤트 조합) |

### 코드 진행 DNA 요약

```
I(add9→M7 하강) → VI7(Secondary Dom) → IIm7 → V(sus4→해결)
= 써클 오브 피프스 + 재즈 확장 화성 + 원-루프 미니멀리즘
```

| **언어 혼용** | Verse=한국어, Chorus=영어 — 자연스러운 코드스위칭 |
