# Lo-fi 장르 심층 리서치

> **작성일:** 2026-03-05
> **목적:** 11-00 시리즈 기획을 위한 Lo-fi 장르 리서치
> **용어 정리:** Classic Lo-fi = 원조 Lo-fi Hip-hop (Nujabes 스타일) ≠ Classical Lo-fi (오케스트라+Lo-fi)

---

## 1. Lo-fi 정의와 기원

### 정의

**Lo-fi (Low Fidelity)** = 일반적으로 녹음의 "결함"으로 여겨지는 요소들이 의도적 미학으로 사용되는 음악/프로덕션 스타일

- vinyl crackle, tape hiss, background noise
- pitch wobble, wow and flutter
- muffled, dusty sound

> *"What professional recordings try to erase, lo-fi music intentionally amplifies."* — MasterClass

### 역사적 변천

| 시대 | 특징 |
|------|------|
| **1950s-80s** | "Low-fi" = 저예산 녹음 기술 설명 |
| **1990s** | 인디 록에서 장르로 정체성 확립 (Guided by Voices, Beck, Pavement) |
| **2000s** | 힙합과 결합 — **Nujabes**, J Dilla가 선구자 |
| **2010s** | YouTube 스트리밍 문화 폭발 (ChilledCow → Lofi Girl) |
| **2020s** | COVID-19로 재택근무/학습 배경음악으로 급성장 (검색량 85%↑) |

### 핵심 인물: Nujabes (Jun Seba)

> *"Godfather of Lo-fi Hip Hop"*

- **Samurai Champloo** (2004) OST로 서양에 Lo-fi 확산
- "Battlecry", "Aruarian Dance" — 바이럴 히트
- 재즈 샘플 + 아날로그 따뜻함 + 붐뱁 드럼
- 사후 (2010) 인터넷 문화 통해 재조명
- modal jazz 샘플 + turntablism + breakbeats

## 2. Lo-fi 핵심 특성

### 프로덕션 요소

| 요소 | 설명 | 도구 |
|------|------|------|
| **Vinyl Crackle** | 레코드판 잡음 | iZotope Vinyl (무료) |
| **Tape Saturation** | 아날로그 테이프 따뜻함/압축 | RC-20 Retro Color |
| **Bit Crushing** | 디지털 해상도 낮춤 → 거친 질감 | XLN Audio RC-20 |
| **Low-pass Filter** | 고음역 롤오프 (8-12kHz) | 모든 DAW |
| **Pitch Wobble** | Wow & Flutter — 몽환적 흔들림 | SketchCassette |

> *"Pitch drift is the best way to create woozy lo-fi textures."*

### BPM & 템포

| 범위 | 특징 |
|------|------|
| **60-80 BPM** | Lo-fi Sweet Spot (명상적) |
| **70-90 BPM** | 일반적 Lo-fi Hip-hop |
| **80+ BPM** | 너무 빠르면 Ambient 아님 |

> *"There's a BPM sweet spot for lo-fi somewhere between 60-80bpm—anything more can feel too intense."*

### 드럼 패턴

- 8-beat 드럼 패턴 (2 bars = 8 beats)
- Kick: 1st, 3rd beat / Snare: 2nd, 4th beat
- Hi-hat: 8th notes로 푸시
- **Humanization 필수**: 그리드에서 살짝 벗어난 타이밍 (로봇 느낌 제거)

### 코드 진행

- Jazz chord progressions (7th, 9th, 13th)
- 재즈 샘플링 또는 라이브 연주
- 피아노, 로즈, 기타 중심

### 비주얼 미학 (Lofi Girl)

- 일본 애니메이션 스타일
- 90년대 레트로 인테리어 (CRT 모니터, 나무 책장)
- 비 오는 창가, 도시 야경
- 공부/작업하는 인물

## 3. Lo-fi 서브장르 비교

### 핵심 분류

```
Lo-fi (미학/프로덕션)
├── Lo-fi Hip-hop (dusty, muffled, nostalgic) ★ Classic Lo-fi
│   ├── Chillhop (hi-fi, clean, jazz-forward)
│   ├── Jazzhop (jazz improvisation)
│   └── Study Beats (기능적)
├── Ambient Lo-fi (dreamlike, ethereal)
└── Classical Lo-fi (orchestral + lo-fi) — 별도 서브장르
```

### Lo-fi vs Chillhop 핵심 차이

| 구분 | **Lo-fi Hip-hop (Classic)** | **Chillhop** |
|------|----------------------------|--------------|
| **프로덕션** | Dusty, muffled, degraded | Warm but **crisp**, clean |
| **Fidelity** | Low-fi (의도적 저음질) | **Hi-fi** but warm |
| **느낌** | 먼 방에서 들리는 듯 | 바로 옆에서 연주하는 듯 |
| **재즈 처리** | 샘플 "punishment" (필터링) | Live-sounding, present |
| **대표 아티스트** | Jinsang, Kudasaibeats, Nujabes | L'Indécis, Philanthrope |
| **BPM** | 60-80 | 78-105 |
| **Vinyl crackle** | 많이 사용 | 적게 사용 |

> *"Chillhop feels 'present' and 'bright' rather than distant and nostalgic."*
> *"Lofi jazz samples are usually sampled from old records and then 'punished' with effects."*

### Ambient Lo-fi

- 비트보다 사운드스케이프 중심
- 긴 evolving 구성
- 명상/introspection 용도
- 더 느린 페이스 (50-70 BPM)

### Classical Lo-fi (참고용 — 별도 서브장르)

- 피아노/현악 + Lo-fi 프로덕션
- Ólafur Arnalds, Nils Frahm, Max Richter 영향
- Reflective, contemplative mood
- 11-00에서 사용하지 않음

## 4. Classic Lo-fi 프로덕션 가이드

### 필수 요소

| 요소 | 설명 |
|------|------|
| **Dusty/Muffled** | Low-pass filter로 고음역 제거 |
| **Vinyl Crackle** | 레코드 잡음 레이어 |
| **Tape Warmth** | Saturation + 미세 압축 |
| **Pitch Wobble** | Wow & Flutter 효과 |
| **Jazz Samples** | 재즈 코드/악기 샘플링 |
| **Slow Tempo** | 60-80 BPM |
| **Humanized Drums** | 그리드 벗어난 타이밍 |

### 금지 요소 (Chillhop과 구분)

- ❌ Hi-fi, crisp, clean 프로덕션
- ❌ Present, bright 느낌
- ❌ 과도하게 빠른 BPM (85+)
- ❌ EDM, aggressive drums

### 분위기 키워드

- Nostalgic, dreamy, hazy
- Distant, muffled, dusty
- Relaxed, meditative, introspective
- Late night, rainy, cozy

## 5. 11-00 시리즈 적용 방향

### 10-00 vs 11-00 포지셔닝

| 구분 | 10-00 (Chillhop) | **11-00 (Classic Lo-fi)** |
|------|------------------|---------------------------|
| Fidelity | Hi-fi, clean | **Lo-fi, dusty** |
| 느낌 | Present, bright | **Distant, nostalgic** |
| 프로덕션 | Warm but crisp | **Muffled, vinyl crackle** |
| BPM | 78-105 | **60-80** |
| 시간대 | 업무 시작 | **점심 직전, 오전 마무리** |

### 감정 방향 제안

- 11:00 = 오전 막바지, 점심 전 허기
- 몽환적, 흐릿한 집중
- "조금만 더" 마무리 감성
- 시계를 슬쩍 보는 순간
