#### 장르 상세 설명

**Rock 계열:**
- **Punk Rock**: 단순하고 거친 코드 진행과 반항적 태도
- **Grunge**: 너바나로 대표되는 90년대 얼터너티브 락
- **Shoegaze**: 기타 이펙터를 과도하게 사용하여 몽환적인 소음의 벽
- **Post-Rock**: 락 악기를 사용하지만 리듬과 텍스처에 집중하는 연주 중심
- **Math Rock**: 복잡한 박자와 변박을 사용하는 테크니컬한 락
- **Emo**: 감성적이고 내면의 고백을 담은 펑크/락
- **Nu Metal**: 힙합, 펑크 등을 결합한 하이브리드 메탈 (예: 린킨 파크)
- **Psychedelic Rock**: 환각적인 경험을 음악으로 표현한 60-70년대 스타일
- **Soft Rock**: 듣기 편하고 대중적인 멜로디의 락

**전자음악 계열:**
- **Drum and Bass (DnB)**: 매우 빠르고 복잡한 브레이크비트 리듬
- **UK Garage**: 셔플 리듬과 보컬 컷업이 특징인 영국발 댄스 음악
- **IDM (Intelligent Dance Music)**: 감상과 실험에 초점을 맞춘 전자음악
- **Hardstyle**: 강렬한 킥 드럼과 왜곡된 사운드
- **Glitch Hop**: 디지털 오류음(Glitch)을 리듬으로 활용
- **Tropical House**: 밝고 청량한 분위기의 하우스 음악
- **Future Bass**: 몽환적인 신디사이저와 강한 베이스 라인
- **Eurobeat**: 빠른 템포와 신디사이저 멜로디 (이니셜 D OST 등)

**흑인 음악 & 어반:**
- **Soul**: 가스펠과 R&B가 결합된 깊은 감정의 보컬 음악
- **Motown**: 60년대 팝 소울의 대명사
- **Drill**: 어둡고 폭력적인 가사와 독특한 베이스 슬라이딩
- **Grime**: 영국에서 발생한 빠르고 거친 일렉트로닉 힙합
- **Boom Bap**: 90년대 동부 힙합의 투박한 드럼 사운드
- **New Jack Swing**: 80-90년대 댄스 비트와 R&B의 결합

**월드 & 지역별:**
- **K-Pop**: 한국의 대중음악
- **J-Pop / City Pop**: 일본의 대중음악 및 70-80년대 도회적 팝
- **Reggaeton**: 힙합과 라틴 리듬이 섞인 중남미 댄스 음악
- **Bossa Nova**: 재즈와 삼바가 결합된 브라질의 부드러운 음악
- **Afrobeats**: 서아프리카의 현대적 팝/댄스 음악
- **Trot**: 한국의 전통적인 대중가요 장르

**숏폼 & 인터넷 네이티브 (2020년대):**
- **Hyperpop**: 팝 음악의 클리셰를 극단적으로 과장하고 왜곡 (예: 100 gecs, Charli XCX)
- **Phonk**: 90년대 멤피스 랩 보컬 샘플에 어두운 사운드
- **Drift Phonk**: 카우벨(Cowbell) 소리가 강조된 자동차 영상 배경음악
- **Glitchcore**: 디지털 오류음과 빠른 템포, 정신없는 편집
- **Sped Up**: 기존 곡의 속도를 빠르게 높인 스타일 (Nightcore의 현대판)
- **Sigilkore**: 게임 사운드, 트랩, 하이퍼팝이 섞인 인터넷 언더그라운드 랩

**댄스 & 리듬 트렌드:**
- **Jersey Club**: 미국 뉴저지 댄스 음악, '쿵-쿵-쿵쿵쿵' 킥 패턴 (K-pop에 많이 차용)
- **Baltimore Club**: 저지 클럽의 원류, 거칠고 빠른 브레이크비트
- **2-Step**: 90년대 영국 장르 부활, 몽환적이면서 잘게 쪼개지는 리듬 (예: NewJeans, PinkPantheress)
- **Breakcore**: 아멘 브레이크 드럼을 복잡하고 빠르게 쪼갠 장르

**글로벌 & 힙합 트렌드:**
- **Amapiano**: 남아공 하우스 음악, 로그 드럼 베이스 + 재즈 건반 (현재 글로벌 핫)
- **Rage**: 비디오 게임 같은 신디사이저와 강력한 베이스 (예: Playboi Carti, Yeat)
- **Pluggnb**: 몽환적 신디사이저 + R&B 보컬과 랩
- **Jersey Drill**: 드릴 비트 + 저지 클럽 댄스 리듬

**분위기 & 미학:**
- **Bedroom Pop**: 침실에서 혼자 만든 듯한 Lo-fi한 질감의 인디 팝
- **Weirdcore / Dreamcore**: 기괴하고 노스텔지어 가득한 음악 스타일

## 2. Style Prompt 조합 공식

### 핵심 원리

> **AI는 앞에 있는 요청부터 반영해서 뒤의 요청을 생성하기 때문에, 뒤로 갈수록 반영이 잘 되지 않을 가능성이 높다.**

### 공식

```
[Major/Minor] + [Genre] + [Tempo] + [Musical instrument include Vocal] + [Mood]
```

### 예시

**조합:**
- `Minor` + `Lo-fi` + `60BPM` + `Piano & Vinyl crackle` + `Whispery female vocals` + `Nostalgic atmosphere` + `Late night vibes`

**완성된 프롬프트:**
```
Minor, Lo-fi hiphop, 60bpm, Melancholic piano keys with vinyl crackle sound, Whispery female vocals, Nostalgic atmosphere, Late night vibes,
```

### 팁

| 상황 | 배치 |
|------|------|
| 프롬프트가 짧을 때 | 장조/단조를 앞에 |
| 프롬프트가 길어지면 | 장르를 앞에 |

## 3. 인기 있는 구조 공식

| 구조 | 패턴 | 특징 및 용도 | 대표곡 |
|------|------|-------------|--------|
| **Pop Standard** | V-C-V-C-B-C | 가장 대중적. 안정감 → 브릿지 → 마지막 코러스 감동 | 아이유 밤편지 |
| **Storyteller** | V-V-C-V-C | 포크/스토리텔링용. Verse 공간 많음, Chorus는 주제 상기 | 마이클 잭슨 빌리진 |
| **Short & Sweet** | C-V-C-V-C | 브릿지 없이 짧고 강렬. 힙합에서 많이 사용 | Dr.dre Still dre, Not like us |
| **K-POP Standard** | V-P-C-V-P-C-B-C | 케이팝 대표 문법. Pre-Chorus로 고조 → Chorus 터뜨림 | Blackpink, 아이유 너랑나 |
| **K-POP 변형 (힙합)** | Hook-V-P-Hook-V-P-Hook-B-Hook | 훅부터 시작. 케이팝 힙합 대표 | 박재범 몸매 |

## 4. 메타태그 in Lyrics: 곡의 구조에 변화를 주는 방법

### 핵심 규칙

| 태그 형태 | 동작 | 설명 |
|----------|------|------|
| `[AA]` | **절대 가사로 읽지 않음** | 구조 태그용 |
| `(BB)` | **조건부** | 코러스/애드립 가사 = 100% 읽음, 톤/악기 지시 = ~30% 인식 |

### 소괄호 `()` 사용 규칙 — Wavvy 검증 (2026-02)

| 규칙 | 설명 |
|------|------|
| **위치** | 모든 구조 태그 (`[Intro]`, `[Verse]`, `[Chorus]` 등) 다음 줄에 **단독 배치** (필수) |
| **포함 가능** | 보컬, 코드 진행, 악기, 악기 진행 |
| **포함 불가** | Exclude 항목 (Exclude는 Style Prompt에만) |
| **같은 맥락** | 공백으로 연결 (쉼표/+ 없이) |
| **다른 맥락** | 쉼표 `,` 또는 `+`로 구분 |

**올바른 예시:**
```
[Intro]
(Instrumental, Male Vocal speaking)

[Verse1]
(Male Vocal soft whisper + Piano arpeggios gentle)
가사 내용...

[Chorus]
(warm major9 and 13 chords with emotional release)
가사 내용...
```

**잘못된 예시:**
```
[Verse1]
가사 내용 (soft whisper)  ← 가사 중간에 넣으면 가사로 읽힘!
```

### 주의사항

```
❌ [sad, emotional piano intro] → AI가 이해하지 못함
❌ 가사 중간에 () 삽입 → 가사로 읽힘
✅ 기분/느낌은 Style에 적으세요
✅ ()는 구조 태그 바로 다음 줄에 단독 배치
```

## 6. 기타: 음악과 관련된 프롬프트 모음

### 6.1 구조 & 길이

```
Make the intro longer
Make the intro longer and more lyrical
Write a longer and more impactful outro
Let the outro fade slowly and emotionally
Add a short instrumental break
Strip the arrangement down in the outro
```

### 6.2 공간감 / 잔향 관련

```
Add subtle room reverb
Create a small jazz club atmosphere
Keep the reverb soft and natural
Make the space feel intimate rather than wide
Let the sound feel close and personal
```

### 6.3 다이내믹 & 밀도

```
Let the mix breathe
Leave space between instruments
Keep the arrangement spacious
Reduce density in the middle section
Build intensity gradually toward the end
```

### 6.4 악기 분리감 & 레이어링

```
Keep clear separation between instruments
Make each instrument feel distinct
Avoid cluttered arrangements
Focus on clarity rather than loudness
```

## Wavvy 적용 포인트

### Style Prompt 작성 시

1. **핵심을 앞에** - Genre/BPM을 첫 3-5단어에 배치 (§2 조합 공식)
2. **구체적 묘사 우선** - 추상적 장르 라벨보다 악기+리듬+보컬 묘사
3. **공간감 프롬프트 활용** - §6.2~6.4 문장 1-2개 추가

### 가사 디자인 시

1. **구조 공식 참조** - §3 인기 있는 구조 공식
2. **메타태그 규칙 준수** - §4 `[AA]`/`(BB)` 동작 원리
3. **톤 지시는 단독 행** - 구조 태그 아래에 별도 배치

### 곡 수정 시

1. **발음만 틀린 경우** - Cover 기능 (Audio 100%, Style 100%)
