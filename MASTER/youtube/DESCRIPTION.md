# YouTube Description Template (SSOT v1.0)

Version: 1.0
Last Updated: 2026-03-03
Purpose: Wavvy 채널 공식 설명 템플릿

---

## 1. 3단 구조

```
1. 인트로 (2-3줄) — 감정적 훅 + 브랜드
2. Track List — 타임스탬프 챕터
3. 아웃트로 — 채널 소개 + 저작권 + 해시태그
```

---

## 2. 인트로 템플릿 (2줄 완결)

> 모바일 "더보기" 전 2줄만 노출

```
{EMOJI} {HH}:{MM}, {TIME_HOOK_SHORT}
{BRAND_MESSAGE} - Wavvy
```

**예시 (06:00):**
```
🌅 06:00, 잠에서 몸이 먼저 깨어나는 시간.
저역 위로 하루를 시작해요. - Wavvy
```

---

## 3. 확장 훅 + Korean Positioning

"더보기" 이후:
```
{EXTENDED_HOOK}
Wavvy는 우리말 가사로, 그 시간들의 감정을 기록합니다.
```

## 6. 해시태그 가이드

> 검색량 높은 태그 우선. 티어별 개행.

| Tier | 목적 | 예시 |
|------|------|------|
| 장르 | 음악 스타일 | `#chillhop #lofi #jazzhop` |
| 상세 목적 | TPO/용도 | `#노동요 #집중 #모닝루틴` |
| 채널 목적 | 발견성 | `#playlist #플리 #음악추천` |
| 브랜드 | 채널 고유 | `#wavvy #웨이비` |

---

## 7. 태그 가이드 (업로드 시)

> YouTube Studio 태그 입력란용. # 없이, 쉼표 구분.

**예시:**
```
chillhop, lofi, fast lofi, 노동요, 모닝루틴, playlist, 플리, wavvy, 웨이비
```

**공통 태그:**
```
playlist, 플리, 음악추천, wavvy, 웨이비, 한국어가사, kpop
```
