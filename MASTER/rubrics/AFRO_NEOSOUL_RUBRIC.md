# K-Afro Neo-Soul Rubric

Version: 1.0
Last Updated: 2026-03-08
Purpose: K-Afro Neo-Soul 장르 품질 게이트

---

## Hard Gates (1개라도 FAIL = 재작성)

| # | Gate | 기준 |
|---|------|------|
| H1 | BPM | 100-115 범위 |
| H2 | Lead | Kalimba/Marimba melody 또는 Rhodes/Neo-Soul keys 포함 |
| H3 | Rhythm | African percussion 포함 |
| H4 | Vocal | 한국어 가사 + R&B 톤 |
| H5 | Energy | Groovy but not aggressive |

---

## 6-Factor Scoring (100점)

| # | Factor | 배점 | 기준 |
|---|--------|------|------|
| F1 | Afro Groove | 20 | African percussion, shaker swing, log drum texture |
| F2 | Neo-Soul Harmony | 20 | Maj7/m7 코드, Rhodes, warm progression |
| F3 | 한국어 가사 | 15 | 대화체, 이미지 중심, 직접 감정 표출 금지 |
| F4 | 에너지 아크 | 15 | V→V2→C 상승, Chorus에 groovy peak |
| F5 | Groovy 밸런스 | 15 | Danceable but chill, 점심 분위기 |
| F6 | 트렌드 터치 | 15 | Amapiano 요소 (subtle), 2025-26 사운드 |

---

## 판정

| 점수 | 판정 | 액션 |
|------|------|------|
| 85+ | PASS | 진행 |
| 70-84 | BORDERLINE | 수정 후 재평가 |
| <70 | FAIL | 재작성 |

**CRITICAL FAIL:** 개별 Factor ≤5점 = 즉시 FAIL

## Style Checklist

| # | 항목 | 체크 |
|---|------|------|
| S1 | BPM 100-115 | ☐ |
| S2 | Kalimba/Marimba + Rhodes (또는 Neo-Soul keys) | ☐ |
| S3 | African percussion / shaker | ☐ |
| S4 | Log drum texture (subtle) | ☐ |
| S5 | Maj7/m7 chord progression | ☐ |
| S6 | Warm vocal (R&B) | ☐ |
| S7 | Groovy, uplifting 키워드 | ☐ |
| S8 | Exclude: EDM drops, heavy 808 | ☐ |
| S9 | Exclude: Falsetto, airy | ☐ |
| S10 | Articulation 포함 | ☐ |

## 가사 Checklist

| # | 항목 | 체크 |
