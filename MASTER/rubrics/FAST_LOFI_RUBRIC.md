# Fast Lo-fi Genre Rubric

Version: 1.4
Last Updated: 2026-03-03
Pass: ≥80점 | Critical Fail: 개별 ≤5점
Evidence: `rubrics/evidence/FAST_LOFI_EVIDENCE.md`

---

## 핵심 철학

> **Fast Lo-fi = 저역 드라이브가 있는 몰입형 그루브**
> lo-fi 질감은 배경, 추진력은 드럼/베이스가 책임.

---

## Hard Gates (FAIL 시 채점 전 재작성)

1. Style 첫 단어 **`Articulation`**
2. BPM **110-150** 명시
3. Exclude 1-8개 + 하모니 가드 2개+
4. 가사 구조 태그와 `()` 메타 **영어**
5. 목표 길이 **3분+** 명시

## 6-Factor Rubric (100점)

### 1. Tempo & Drive — 20점
| 18-20 | BPM 122-128 중심, 러닝/워크아웃 몰입 |
| 14-17 | 110-150 준수, 일부 추진력 약화 |
| 10-13 | 템포 체감 느슨 |
| ≤9 | 추진력 부족 |

**범위**: 110-150, 중심 122-128, 피크 140-150

### 2. Drum Impact — 20점
| 18-20 | 킥 타격감 선명, 스네어 어택 명확, 하이햇 속도감 |
| 14-17 | 존재감 양호, 일부 어택 약함 |
| 10-13 | 밀도/임팩트 부족 |
| ≤9 | 드럼 드라이브 부재 |

**원칙**: punchy kick + tight snare + fast hats

### 3. Low-end Clarity — 20점
| 18-20 | sub + mid bass 이중 구조, 번짐 없음 |
| 14-17 | 저역 양호, 일부 탁함 |
| 10-13 | 윤곽 불명확 |
| ≤9 | 킥/베이스 충돌, muddiness |

**원칙**: sub(무게) + mid-bass(그루브) 분리

### 4. Lo-fi Texture Balance — 15점
| 14-15 | 노이즈/빈티지 질감이 배경에 머묾 |
| 11-13 | 특정 구간 과함 |
| 7-10 | 본체와 분리 안 됨 |
| ≤6 | 질감이 드럼/베이스 가림 |

**원칙**: lo-fi는 분위기 레이어

### 5. Structural Runtime — 15점
| 14-15 | 자연스러운 전개, 3분+ 달성 |
| 11-13 | 밀도 단조 |
| 7-10 | 반복감 높음 |
| ≤6 | 길이 미달/구조 부재 |

**권장 섹션**: intro/verse/pre-chorus/chorus/interlude/bridge/outro

### 6. Genre Boundary Guard — 10점
| 9-10 | Fast Lo-fi 정체성 명확 |
| 7-8 | 일부 경계 혼합 |
| ≤6 | Hip-hop/EDM/Rock 애매 |

**금지**: EDM supersaw, trance arp, festival riser, rock drums

## 체크리스트

```
□ BPM 110-150, 중심 122-128
□ 킥/스네어/하이햇 추진력
□ 저역(sub+mid) 선명
□ lo-fi 질감이 메인 가리지 않음
□ 3분+ 구조 전개
□ "Fast Lo-fi" 즉시 설명 가능
```
