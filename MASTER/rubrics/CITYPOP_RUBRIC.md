# City Pop Genre Rubric

Version: 1.1
Last Updated: 2026-03-03
Pass: ≥80점 | Critical Fail: 개별 ≤5점

---

## 핵심 철학

> **시티팝 = 도시를 '쿨하게' 기억하는 방식**
> 장르라기보다 '도시를 대하는 태도와 문법의 묶음'.

---

## 6-Factor Rubric (100점)

### 1. Urban Lyricism (도시 서사) — 20점
| 18-20 | 도시 오브젝트 4종+, "도시" 명시, 다중 감각, "세련된 외로움" |
| 14-17 | 2-3종, 도시임 알 수 있음 |
| 10-13 | 1-2종, 도시/자연 구분 애매 |
| ≤9 | 도시 요소 부재/자연 중심 |

**오브젝트**: 네온사인, 가로등, 교각, 택시, 아파트, 강변도로, 칵테일, 드라이브

### 2. Cool Restraint (감정 절제) — 15점
| 14-15 | 과잉 제로, 관조적, "그냥 남아 있다" |
| 11-13 | 대체로 절제, 1-2곳 소망 표현 |
| 7-10 | 절제/과잉 혼재 |
| ≤6 | 감정 폭발, 고음 클라이맥스 |

**원칙**: ❌ 고음 폭발, "아프다" → ⭕ "그냥 남아 있다"

### 3. Tension Harmony (텐션 화성) — 20점
| 18-20 | Maj7/m7/9th/sus/slash + IV→iv + "붕 뜬 몽환감" |
| 14-17 | 확장 코드 3종+, IV→iv 또는 Secondary Dom |
| 10-13 | 확장 1-2종, 시그니처 없음 |
| ≤9 | 단순 I-V-vi-IV 팝 코드 |

**시그니처**: IV→iv 감정 전이

### 4. Funky Groove (리듬/베이스) — 15점
| 14-15 | 슬랩/펑키 핑거, 16비트 그루브, BPM 적합 |
| 11-13 | 펑키 있으나 BPM 경계선 |
| 7-10 | 베이스 있으나 펑키 아님 |
| ≤6 | 베이스 존재감 없음 |

**BPM별 베이스**:
- 100+: Slap bass
- 90-100: Funky finger
- 80-90: Mellow round finger (슬랩 주의)

### 5. Vintage Sound (사운드 팔레트) — 20점
| 18-20 | 7요소 중 6-7개 |
| 14-17 | 4-5개 |
| 10-13 | 2-3개 |
| ≤9 | 1개 이하 |

**7요소**: Juno synth, DX7 FM, Brass, 코러스 클린 기타, Plate Reverb, Tape warmth, 라이브 드럼

**주의**: ❌ Dry/Raw → ⭕ Lush plate reverb + 공간감

### 6. Genre Identity — 10점
| 9-10 | "시티팝이다" 즉시 인식 |
| 7-8 | 1개 요소 약함 |
| ≤6 | R&B/발라드/재즈 애매 |

---

## 체크리스트

```
□ 도시가 감정의 주체다
□ 감정이 절제되어 있다
□ 재즈/소울 화성 문법
□ 80s 일본 팝 사운드 계보
□ "시티팝이다" 방어 가능
```

> 3개 이상 = 시티팝, 4개 = 확실, 5개 = 정통
