## Evidence Basis

### 1. Classic Lo-fi 정의
- Lo-fi (Low Fidelity) = 녹음의 "결함"을 의도적 미학으로 사용
- Vinyl crackle, tape hiss, pitch wobble, muffled sound
- 대표 아티스트: Jinsang, Kudasaibeats, idealism, Lofi Girl
- 2010년대 YouTube 스트리밍 문화로 대중화

### 2. Chillhop vs Classic Lo-fi 핵심 차이
| 구분 | Chillhop | Classic Lo-fi |
|------|----------|---------------|
| 프로덕션 | Warm but **crisp**, clean | Dusty, **muffled**, degraded |
| Fidelity | Hi-fi but warm | **Lo-fi** (의도적 저음질) |
| 느낌 | Present, bright | **Distant, nostalgic** |
| 재즈 처리 | Live-sounding | Sampled + **"punished"** |
| BPM | 78-105 | **70-90** |
| Vinyl crackle | 적게 사용 | **많이 사용** |

> *"Chillhop feels 'present' and 'bright' rather than distant and nostalgic."*
> *"Lofi jazz samples are usually 'punished' with effects."*

### 3. 핵심 프로덕션 요소
| 요소 | 설명 |
|------|------|
| Vinyl Crackle | 레코드판 잡음 |
| Tape Saturation | 아날로그 테이프 따뜻함/압축 |
| Bit Crushing | 디지털 해상도 낮춤 → 거친 질감 |
| Low-pass Filter | 고음역 롤오프 (8-12kHz) |
| Pitch Wobble | Wow & Flutter — 몽환적 흔들림 |

### 4. BPM & 템포
| 범위 | 특징 |
|------|------|
| **70-90 BPM** | Lo-fi Sweet Spot (일반적) |
| 70-80 BPM | 느린 Lo-fi (명상적) |
| 80-90 BPM | 표준 Lo-fi Hip-hop |

> *"Lo-fi hip hop typically ranges from 70-90 BPM with muffled, dusty production."*

### 5. 드럼 패턴
- 8-beat 기본 (2 bars = 8 beats)
- Kick: 1st, 3rd beat / Snare: 2nd, 4th beat
- Hi-hat: 8th notes로 푸시
- **Humanization 필수**: 그리드에서 ±10-20ms 오프셋 (로봇 느낌 제거)

### 6. 코드 진행 & 악기
- Jazz chord progressions (7th, 9th, 13th)
- 재즈 샘플링 또는 라이브 연주
- **핵심 악기**: Piano, Rhodes, Guitar
- **베이스**: Deep, muffled upright bass
- **Lo-fi 특성**: Sampled jazz + muffled drums + vinyl texture

### 7. 분위기 키워드
| 허용 | 금지 |
|------|------|
| Nostalgic, dreamy, hazy | Crisp, clean, bright |
| Distant, muffled, dusty | Present, polished |
| Relaxed, meditative, hypnotic | Energetic, aggressive |
| Late night, rainy, cozy, warm | Upbeat, cheerful |

## Hard Gates (FAIL 시 채점 전 재작성)

1. BPM **70-90** 명시
2. Lo-fi 프로덕션 요소 **3개+** (vinyl crackle, low-pass filter, tape saturation, pitch wobble, bit crushing)
3. **"dusty"** 또는 **"muffled"** 키워드 포함
4. A/B: Vocal buried + distant / C: Instrumental + texture 명시
5. Exclude 1-8개 + 하모니 가드 2개+
6. 목표 길이 **2분30초+** 명시
7. **"crisp", "clean", "bright", "present"** 키워드 미포함
8. [Intro] = **Instrumental only**
9. Humanized drums 명시

## 6-Factor Rubric (100점)

### 1. Lo-fi Aesthetics — 25점

| 23-25 | Dusty + muffled + vinyl crackle + pitch wobble 완벽 |
| 18-22 | Lo-fi 요소 3개+, 일부 약함 |
| 12-17 | Lo-fi 요소 2개, Chillhop에 가까움 |
| ≤11 | Hi-fi / clean / Chillhop |

**필수 요소:**
- Vinyl crackle layer
- Low-pass filter (8-12kHz 롤오프)
- Tape saturation
- Pitch wobble (Wow & Flutter)
- Reverb/delay (spatial enhancement)
- Bit crushing (optional)

**금지:** Crisp, clean, bright, present, hi-fi

### 2. Jazz Foundation — 15점

| 14-15 | Jazz chords (7th/9th/13th) + 재즈 악기 2개+ |
| 11-13 | 재즈 요소 2개, 샘플 "punished" 처리 |
| 7-10 | 재즈 요소 1개 |
| ≤6 | 재즈 없음 |

**필수 악기:** Piano, Rhodes, Guitar, Upright Bass
**처리:** 샘플은 필터링/degrading 필수 ("punishment")

### 3. Groove & Feel — 15점

| 14-15 | Boom-bap groove, humanized, 70-90 BPM |
| 11-13 | 그루브 양호, 일부 기계적 |
| 7-10 | Swing feel 부족 / BPM 초과 |
| ≤6 | 4-on-the-floor / EDM 느낌 |

**드럼 패턴:**
- 8-beat 기본
- Kick: 1st, 4th, 7th (변형 가능)
- Snare: 3rd, 7th (백비트)
- Hi-hat: 8th notes, swing 허용
- **Humanization 필수:** 그리드 ±10-20ms 오프셋

### 4. Vocal/Texture Integration — 20점

**A/B (보컬):**
| 18-20 | 보컬이 먼 방에서 들리듯, 악기에 완전히 묻힘 |
| 14-17 | 묻혀 있지만 가끔 튀어나옴 |
| ≤13 | 보컬이 분리/전면 |

**C (인스트):**
| 18-20 | 텍스처 레이어 풍성, 허밍/스캣이 배경에 녹아듦 |
| 14-17 | 텍스처 있지만 단조로움 |
| ≤13 | 텍스처 없거나 튀어나옴 |

**원칙:** 보컬 = 또 하나의 악기, 절대 전면에 안 나옴

### 5. Structural Flow — 15점

| 14-15 | 루프 기반, 부드러운 전환, 2분30초+ |
| 11-13 | 전환 다소 급격 / 길이 살짝 미달 |
| ≤10 | 반복감 높음 / 구조 불분명 |

**구조:**
- [intro] → [loop A] → [loop B] → [loop A'] → [outro]
- 인터루드 활용 가능
- 급격한 전환 금지

### 6. Genre Boundary Guard — 10점

| 9-10 | Classic Lo-fi 정체성 명확, Chillhop과 구분됨 |
| 7-8 | 일부 Chillhop 요소 혼합 |
| ≤6 | Chillhop / Ambient / R&B 애매 |

**금지:**
- EDM, aggressive drums
- 4-on-the-floor 킥 패턴
- Bright/present 프로덕션
- 90+ BPM

## 체크리스트

```
□ BPM 70-90
□ Lo-fi 프로덕션 3개+ (vinyl, low-pass, tape, wobble)
□ "dusty" 또는 "muffled" 키워드 포함
□ "crisp", "clean", "bright" 미포함
□ Jazz 요소 2개+ (샘플 punished 처리)
□ Humanized drums (±10-20ms)
□ 보컬 buried / 인스트 텍스처 처리
□ 2분30초+, 루프 기반 구조
□ "Classic Lo-fi"라고 즉시 설명 가능
```
