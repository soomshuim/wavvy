## Evidence Basis

### Chillhop vs Lo-fi Hip Hop 차이점

| 항목 | Lo-fi Hip Hop | Chillhop |
|------|---------------|----------|
| **BPM** | 60-90 | **80-110** |
| **음질** | Low-fidelity, dusty, muffled | **High-fidelity, crisp, polished** |
| **악기** | Filtered samples, "underwater" | **Live-sounding: Rhodes, sax, acoustic guitar** |
| **분위기** | 밤, 고독, 수면용 | **낮, 활기, 작업/공부용** |
| **질감** | Vinyl crackle, tape hiss 강함 | Warm하지만 깨끗 |

### Chillhop 핵심 특성
- Lo-fi hip hop + Jazz 요소 결합
- **High-fidelity** production with crisp drums
- **Live-sounding instrumentation**: Rhodes piano, saxophone, acoustic guitar
- Swing feel, jazzy chord voicings (7th/9th chords)
- "Daylight" mood vs Lo-fi's "night" mood

## 핵심 철학

> **Chillhop = 재즈의 세련됨 + 따뜻한 폴리싱**
> Lo-fi의 "dusty/muffled"가 아니라 **"warm but crisp"**.

---

## Hard Gates (FAIL 시 채점 전 재작성)

1. BPM **78-105** 명시
2. Jazzy 요소 **2개+** (Rhodes, Wurlitzer, jazz guitar, sax, upright bass, swing, jazzy chords)
3. A/B: Vocal buried 지시 / C: Instrumental 또는 허밍/스캣 텍스처 명시
4. Exclude 1-8개 + 하모니 가드 2개+
5. 목표 길이 **3분+** 명시
6. "dusty", "muffled" 키워드 **미포함**
7. [Intro] = **Instrumental only**

## 6-Factor Rubric (100점)

### 1. Jazz Elements — 20점
| 18-20 | 재즈 코드(7th/9th) + 스윙 + 재즈 악기 2개+ |
| 14-17 | 재즈 요소 2개, 일부 약함 |
| 10-13 | 재즈 요소 1개만 |
| ≤9 | Lo-fi에 가까움 |

**필수 악기**: Rhodes, Wurlitzer, Jazz Guitar, Sax, Upright Bass

### 2. Groove & Feel — 20점
| 18-20 | Laid-back swing, "daylight" 에너지 |
| 14-17 | 그루브 양호, 일부 어색 |
| 10-13 | Swing feel 부족 |
| ≤9 | 기계적/straight beat |

**원칙**: Swing feel, lazy groove, "깔아두는 음악"

### 3. Warmth & Polish — 15점
| 14-15 | High-fidelity but warm, subtle saturation |
| 11-13 | 약간 muddy 또는 차가움 |
| 7-10 | Dusty/muffled 또는 sterile |
| ≤6 | 음질 문제 심각 |

**원칙**: ✅ Crisp, clean ❌ Dusty, muffled, heavy vinyl

### 4. Vocal/Texture Integration — 20점

**A/B (보컬):**
| 18-20 | 보컬이 악기처럼 묻혀 배경 텍스처화 |
| 14-17 | 살짝 앞으로 나옴 |
| ≤13 | 보컬이 분리/전면 |

**C (인스트):**
| 18-20 | 레이어 풍성, 허밍/스캣 자연스럽게 녹아듦 |
| ≤13 | 단조롭거나 텍스처 튀어나옴 |

### 5. Structural Flow — 15점
| 14-15 | 부드러운 전환, 3분+ 달성 |
| 11-13 | 전환 다소 급격 |
| ≤10 | 반복감 높음/길이 미달 |

### 6. Genre Boundary Guard — 10점
| 9-10 | Chillhop 정체성 명확 |
| 7-8 | 일부 경계 혼합 |
| ≤6 | Lo-fi/Jazz/R&B 애매 |

**금지**: EDM, aggressive drums, rock energy, heavy distortion

## 체크리스트

```
□ BPM 78-105
□ 재즈 요소 2개+ (Rhodes/Sax/Jazz Guitar/Swing)
□ High-fidelity but warm (dusty 아님)
□ 보컬 buried / 인스트 텍스처 처리
□ 3분+, 부드러운 전환
□ "Chillhop"이라고 즉시 설명 가능
```
